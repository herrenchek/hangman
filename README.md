# Hangman Game (Browser)
Word guess game commonly know as 'hangman', uses JavaScript to dynamically update web page. Inspired by 16-bit games and the Alien franchise. Refactored code to use ES6 standard.

## Built With
`HTML`
`CSS`
`JavaScript`
`Bootstrap`
